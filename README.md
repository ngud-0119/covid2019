# COVID-19 :mask: API

### A GraphQL Express Server powered by [COVID-19 API]() to provide COVID-19 related stats for India :india:



## Queries

#### GET overall COVID-19 stats

```graphql
{ 
  total {
    active
    confirmed
    deaths
    recovered
  }
}
```

#### GET statewise COVID-19 stats

```graphql
{ 
  total {
    active
    confirmed
    deaths
    recovered
  }
}
```

#### GET datewise COVID-19 stats

```graphql
{ 
  total {
    active
    confirmed
    deaths
    recovered
  }
}
```
